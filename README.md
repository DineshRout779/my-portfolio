## Tech Stack
- React
- Styled components

## How to use
1. Clone the repo
``` bash
git clone https://github.com/DineshRout779/my-portfolio
```

2. Install dependencies
``` bash
npm install
```

3. Start the dev server
``` bash
npm start
```
